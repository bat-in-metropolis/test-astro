## ğŸ“ How It Works

1. **Original HTML files** are stored in `static-html/` folder (completely untouched)
2. **Next.js pages** in `pages/` folder read the HTML content at build time
3. **Static generation** happens during build - Next.js generates optimized static HTML
4. **CSS** is automatically loaded via `_app.js`

## ğŸ”§ Adding New Pages

To add a new HTML page:

1. Add your HTML file to `static-html/` folder (e.g., `new-page.html`)
2. Create a corresponding Next.js page in `pages/` folder:

```javascript
// pages/new-page.js
import Head from 'next/head'
import { getStaticHTML } from '../lib/htmlLoader'

export default function NewPage({ bodyContent, title }) {
  return (
    <>
      <Head>
        <title>{title}</title>
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      </Head>
      <div dangerouslySetInnerHTML={{ __html: bodyContent }} />
    </>
  )
}

export async function getStaticProps() {
  const { bodyContent, title } = getStaticHTML('new-page.html')
  return {
    props: {
      bodyContent,
      title,
    },
  }
}
```
